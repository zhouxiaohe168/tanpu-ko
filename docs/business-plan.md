# 探铺 - 连锁品牌智能选址数据平台

> 品牌名称：**探铺**  
> Slogan：**连锁品牌智能选址，一键发现黄金空白点**  
> 域名：tanpu.com / tanpu.cn

---

## 一、产品定位

探铺是一个连锁品牌智能选址数据平台，帮助加盟商通过数据找到品牌空白点位。

**目标用户：**
- 想开加盟店的创业者
- 连锁品牌总部拓展团队
- 商业地产招商人员
- 加盟中介/咨询公司
